
### Code Search Integration

When researching implementation approaches, use code-search MCP tools to find relevant patterns in the codebase:

- `mcp__code-search__search_code` - Search for code patterns, function definitions, and usage examples
- `mcp__code-search__search_symbols` - Find type definitions, interfaces, and class hierarchies
- `mcp__code-search__get_context` - Get surrounding context for a specific code location

Use code-search BEFORE writing implementation to discover existing patterns, conventions, and utilities that should be reused rather than reinvented.
